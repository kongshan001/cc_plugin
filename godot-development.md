# Godot 游戏开发技能

> Godot 4.x 游戏引擎开发技能

## 项目信息

- **GitHub Stars**: 1-15+ (多个相关项目)
- **技能类别**: 游戏客户端开发 / Godot
- **状态**: ✅ 已调研

---

## 简介

Godot 游戏开发技能为 Claude Code 提供 Godot 4.x 游戏引擎的开发支持，包括 GDScript 编程、Shader 编写、场景管理等功能。

---

## 核心技能列表

### 1. Godot GDScript Patterns

- **功能**: GDScript 最佳实践和设计模式
- **特点**: 信号系统、场景管理、状态机、资源管理

### 2. Claude-Code-Game-Studios

- **功能**: 完整游戏开发工作室
- **支持引擎**: Godot 4、Unity、Unreal Engine 5
- **包含**: 48 个 AI agents、36 个工作流技能

### 3. Claude Resources (Godot)

- **功能**: Godot 游戏开发自定义 agents 和 skills

---

## 核心功能

| 功能 | 说明 |
|------|------|
| GDScript 编程 | Godot 专用脚本语言 |
| Shader 开发 | 着色器编写 |
| 场景管理 | 节点操作和场景切换 |
| 资源管理 | 动态加载和资源打包 |
| 扩展开发 | GDExtension C++ 扩展 |
