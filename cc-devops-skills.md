## 简介

cc-devops-skills 是为 DevOps 工程师（或任何需要部署代码的人）准备的极其详细的技能集。它与验证、生成器、Shell 脚本和 CLI 工具配合使用，为您曾经痛苦地工作的努力任何平台创建高质量的 IaC（基础设施即代码）代码。

即使只作为文档来源下载也值得！

---

## 核心功能

### 🔧 验证工具
| 功能 | 说明 |
|------|------|
| 配置验证 | 验证基础设施配置 |
| 语法检查 | 检查脚本语法 |
| 安全性扫描 | 扫描安全漏洞 |

### 📦 生成器
| 功能 | 说明 |
|------|------|
| 项目模板 | 生成项目脚手架 |
| 配置文件 | 生成各类配置文件 |
| 部署脚本 | 生成自动化部署脚本 |

### 🖥️ Shell 脚本
| 功能 | 说明 |
|------|------|
| 自动化脚本 | 日常任务自动化 |
| 监控脚本 | 系统监控和告警 |
| 备份脚本 | 数据备份和恢复 |

### CLI 工具
| 功能 | 说明 |
|------|------|
| 部署工具 | 一键部署 |
| 管理工具 | 服务器管理 |
| 诊断工具 | 问题诊断和排查 |

## 安装方式

```bash
# 克隆仓库
git clone https://github.com/akin-ozer/cc-devops-skills.git

# 复制到 Claude Code skills 目录
cp -r cc-devops-skills ~/.claude/skills/

# 重启 Claude Code
```
