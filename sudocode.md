# Sudocode 轻量级代理编排工具

> 将 Git 仓库作为分布式上下文数据库的代理编排系统

## 1. 背景需求

Claude Code 等 AI 编码代理在长期任务中容易出现"失忆"问题，多代理协作时上下文管理混乱。

## 2. 目标

提供轻量级代理编排系统，实现：
- 规范驱动开发
- Git 原生问题跟踪
- 多代理协调工作流

## 3. 设计方案

```
┌─────────────────────────────────────────────────────────────┐
│                      Sudocode 架构                            │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐       │
│  │   Specs     │    │   Issues    │    │   Executions│      │
│  │  (需求规范)   │    │   (任务)    │    │   (执行轨迹) │      │
│  └──────┬──────┘    └──────┬──────┘    └──────┬──────┘      │
│         │                   │                   │             │
│         └───────────────────┼───────────────────┘             │
│                             │                                │
│                    ┌────────▼────────┐                       │
│                    │   .sudo 目录    │                       │
│                    │  (Git 原生存储)  │                       │
│                    └────────┬────────┘                       │
│                             │                                │
│                    ┌────────▼────────┐                       │
│                    │    Git Repo    │                       │
│                    │  (版本控制)     │                       │
│                    └─────────────────┘                       │
└─────────────────────────────────────────────────────────────┘
```

## 4. 本地部署

```bash
# 全局安装
npm install -g sudocode

# 初始化项目
cd my-project
sudocode init

# 启动本地服务器
sudocode server

# Claude Code 插件安装
claude plugin marketplace add sudocode-ai/sudocode
claude plugin install sudocode
```
