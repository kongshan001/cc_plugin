# aws-mcp-server AWS 云服务 MCP

> AWS 云服务 MCP 服务器集成

## 1. 背景需求

需要通过自然语言管理 AWS 云服务。

## 2. 目标

提供 AWS 服务 MCP 集成，支持 EC2/S3/Lambda 等。

## 3. 设计方案

- MCP 协议支持
- AWS SDK 集成
- 自然语言控制
- 多服务支持

## 4. 本地部署

```bash
pip install awslabs/aws-mcp-server
```
