# developer-kit 全栈开发工具包

> 133 Stars | Python | 多语言开发工具包

## 1. 背景需求

现代开发涉及多种编程语言和框架，开发者需要统一的工具集来提高效率。

## 2. 目标

提供模块化的 Claude Code 插件系统，包含可重用的技能、Agent 和命令，自动化各类开发任务。

## 3. 设计方案

**核心架构**
- 模块化插件系统
- 多语言技能支持
- 预置开发模式

**支持领域**
| 领域 | 技术栈 |
|------|--------|
| Java | Spring Boot、LangChain4J |
| TypeScript | NestJS、React |
| Python | Python 开发 |
| PHP | WordPress |
| AWS | CloudFormation |
| AI | 常见 AI 开发模式 |

## 4. 本地部署

```bash
# 克隆仓库
git clone https://github.com/giuseppe-trisciuoglio/developer-kit.git

# 进入目录
cd developer-kit

# 选择需要的技能目录
# 例如：python/, java/, typescript/
```
