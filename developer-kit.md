# Developer Kit 多语言开发者工具包

> Java/Spring Boot/TypeScript/Python/PHP 多框架开发者工具集

## 1. 背景需求

现代开发往往涉及多种语言和框架，需要统一的知识库和开发规范。

## 2. 目标

提供覆盖主流开发技术的 Claude Code 技能集。

## 3. 设计方案

- **Java 系**: Spring Boot, LangChain4J
- **前端系**: TypeScript, NestJS, React
- **Python**: 全栈开发
- **Web**: PHP/WordPress
- **云**: AWS CloudFormation

## 4. 本地部署

```bash
git clone https://github.com/giuseppe-trisciuoglio/developer-kit ~/.claude/plugins/developer-kit
```
