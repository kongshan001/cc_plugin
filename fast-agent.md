# Fast-Agent - MCP 代理和工作流定义框架

> ⭐ 3,694 Stars · Python · 工作流自动化

## 1. 背景需求

开发者需要一种简单的方式来定义、提示和测试 MCP 启用的代理和工作流。

## 2. 目标

提供统一的框架来构建和管理 MCP 代理，简化 AI 应用开发。

## 3. 核心功能

- 🔄 工作流定义
- 🧪 代理测试
- 🔌 MCP 集成
- 📝 提示词管理
- 📊 执行监控

## 4. 本地部署

```bash
# 安装
pip install fast-agent

# 使用
from fast_agent import Agent

agent = Agent()
agent.run("your task")
```
