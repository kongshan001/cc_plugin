# Dippy Bash 命令过滤器

> 通过 AST 解析自动批准安全命令，减少权限疲劳

## 1. 背景需求

Claude Code 对每个 bash 命令都请求权限，破坏流状态。

## 2. 目标

自动识别并批准安全命令，阻止危险操作：
- 减少 40% 权限提示
- 自定义拒绝消息引导 AI
- 保持安全边界

## 3. 设计方案

```
┌─────────────────────────────────────────────────────────────┐
│                      Dippy 流程                              │
├─────────────────────────────────────────────────────────────┤
│   Claude Code                                               │
│       │                                                     │
│       ▼                                                     │
│   ┌────────────────────┐                                    │
│   │  Bash Command      │                                    │
│   └─────────┬──────────┘                                    │
│             │                                               │
│             ▼                                               │
│   ┌────────────────────┐                                    │
│   │  Dippy Hook        │                                    │
│   │  (Parable Parser)  │                                    │
│   └─────────┬──────────┘                                    │
│             │                                               │
│      ┌──────┴──────┐                                        │
│      ▼             ▼                                        │
│  ┌───────┐    ┌────────┐                                    │
│  │ Safe? │    │ Unsafe │                                    │
│  │ Auto  │    │ Prompt │                                    │
│  │Approve│    │  User  │                                    │
│  └───────┘    └────────┘                                    │
└─────────────────────────────────────────────────────────────┘
```

## 4. 本地部署

```bash
# Homebrew 安装
brew tap ldayton/dippy
brew install dippy

# 手动安装
git clone https://github.com/ldayton/Dippy.git

# Claude Code 配置 (~/.claude/settings.json)
{
  "hooks": {
    "PreToolUse": [
      {
        "matcher": "Bash",
        "hooks": [{ "type": "command", "command": "dippy" }]
      }
    ]
  }
}
```

## 5. 效果展示

- 性能提升：减少 40% 权限提示
- 解析器：自研 Parable，14,000+ 测试
- 自动批准：复杂管道、链式读取、云检查
