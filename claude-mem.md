# claude-mem 智能记忆系统

> 自动捕获会话内容并生成 AI 摘要

## 1. 背景需求

AI 助手需要记忆会话上下文。

## 2. 目标

提供会话记忆和摘要生成。

## 3. 设计方案

- 自动捕获内容
- AI 摘要生成
- 记忆检索
- 上下文恢复

## 4. 本地部署

```bash
# 需要 Bun 运行时
bun install -g claude-mem
```

## 5. 效果展示

- Worker 端口：37777

## 6. 优缺点

✅ 记忆持久化 ✅ 上下文恢复  
⚠️ 需要 Bun ⚠️ 资源消耗
