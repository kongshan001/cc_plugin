# Claude-Code-Game-Studios - 48 AI 代理游戏开发工作室

> ⭐ 30 Stars · Python · 游戏开发

## 1. 背景需求

游戏开发者需要将 Claude Code 打造成完整的游戏开发工作室，需要多个专业 AI 代理协同工作。

## 2. 目标

提供 48 个专业 AI 代理和 36 个工作流技能，模拟真实游戏工作室的层级结构。

## 3. 核心功能

- 🎮 48 个专业 AI 代理
- 🔧 36 个游戏开发工作流技能
- 🏢 工作室协调系统
- 📐 多人游戏支持
- 🎨 美术资源管理
- 📝 剧本和叙事设计
