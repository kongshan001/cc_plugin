# Memory MCP 服务器

> 知识图谱驱动的持久化记忆系统

## 项目信息

- **官方维护**: Model Context Protocol
- **技能类别**: 开发者工具 / 知识管理
- **状态**: ✅ 官方参考实现

---

## 简介

Memory MCP 服务器是 Model Context Protocol 官方参考实现，提供基于知识图谱的持久化记忆系统。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 知识图谱 | 结构化知识存储 |
| 持久化 | 长期记忆保持 |
| 上下文关联 | 智能关联检索 |
| 会话管理 | 跨会话记忆共享 |

---

## 技术架构

- **核心**: 知识图谱 (Knowledge Graph)
- **协议**: MCP (Model Context Protocol)
- **用途**: 参考实现/教育示例

---

# 官方参考实现，可从以下地址获取

## 应用场景

- **对话历史**: 记住之前的对话内容
- **项目知识**: 积累项目特定知识
- **用户偏好**: 记住用户偏好设置
- **上下文恢复**: 跨会话恢复工作上下文
