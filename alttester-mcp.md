# AltTester MCP - 游戏自动化测试

> Unity/Unreal 游戏自动化测试

## 1. 背景需求

游戏开发需要专业的自动化测试工具来确保游戏质量和减少手动测试工作量。

## 2. 目标

提供 AI 驱动的游戏自动化测试能力，支持 Unity 和 Unreal Engine 游戏。

## 3. 核心功能

- 🎮 Unity 游戏自动化测试
- 🎯 Unreal Engine 游戏自动化测试
- 🤖 AI 驱动的测试编写
- 🔍 游戏对象识别
- 🖱️ UI 测试自动化
- 📸 截图和视觉回归测试

## 4. 本地部署

### MCP 配置

```json
{
  "mcpServers": {
    "alttester": {
      "command": "npx",
      "args": ["-y", "@alttester/mcp-server@latest"]
    }
  }
}
```

### 详细配置

1. 安装 AltTester Desktop
2. 配置 MCP 服务器
3. 连接游戏

## 5. 效果展示

- 官方 Unity/Unreal 测试框架
- AI 扩展支持
- 跨平台测试

## 6. 优缺点

✅ 专业游戏测试
✅ AI 辅助编写测试
✅ 支持主流游戏引擎
