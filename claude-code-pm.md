# Claude Code PM 项目管理技能

> 全面强大的 Claude Code 项目管理框架

## 1. 背景需求

现代软件开发需要系统化的项目管理流程。Claude Code 虽然能帮助编写代码，但缺乏完整的项目管理能力。Claude Code PM 提供了一套完整的项目管理技能，让 AI 能够协助管理项目全生命周期。

## 2. 目标

为 Claude Code 提供专业级的项目管理能力，包括任务规划、进度跟踪、团队协作和发布管理。

## 3. 设计方案

### 3.1 核心功能

- **任务管理**: 创建、分配、跟踪任务
- **进度跟踪**: 可视化项目进度
- **里程碑管理**: 关键节点追踪
- **团队协作**: 多角色协调
- **发布管理**: 版本规划和发布流程

### 3.2 技能组成

| 组件 | 功能 |
|------|------|
| 多个专业代理 | 不同项目管理角色 |
| Slash 命令 | 快速执行管理操作 |
| 文档模板 | 标准项目管理文档 |
| 工作流 | 自动化流程 |
