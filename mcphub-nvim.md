# McpHub Neovim - Neovim MCP 客户端

> ⭐ 1,739 Stars · Lua · 编辑器集成

## 1. 背景需求

Neovim 用户需要无缝集成 MCP 服务器到编辑工作流中。

## 2.目标

提供直观的 MCP 客户端界面，用于在 Neovim 中管理、测试和使用 MCP 服务器。

## 3. 核心功能

- 🔌 MCP 服务器管理
- 🧪 服务器测试
- 💻 编辑器内集成
- 📋 直观界面

## 4. 本地部署

```bash
# 使用 packer.nvim
use 'ravitemer/mcphub.nvim'

# 或使用 lazy.nvim
{'ravitemer/mcphub.nvim'}
```
