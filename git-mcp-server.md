# Git MCP 服务器

> 官方 Git 版本控制 MCP 服务器

## 项目信息

- **官方维护**: Model Context Protocol
- **技能类别**: 开发者工具 / 版本控制
- **状态**: ✅ 官方参考实现

---

## 简介

Git MCP 服务器是 Model Context Protocol 官方参考实现之一，提供 Git 仓库的读取、搜索和操作能力。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 仓库读取 | 读取 Git 仓库内容 |
| 历史搜索 | 搜索提交历史 |
| 分支操作 | 管理分支 |
| 差异比较 | 比较不同版本 |

---

## 技术架构

- **SDK**: TypeScript SDK
- **协议**: MCP (Model Context Protocol)
- **用途**: 参考实现/教育示例
