# MCP Inspector - 官方 MCP 服务器可视化测试工具

> ⭐ 8,900 Stars · TypeScript · 官方维护

## 1. 背景需求

MCP 服务器开发者和使用者需要一个可视化工具来测试、调试和验证 MCP 服务器的实现是否符合协议规范。

## 2. 目标

提供官方维护的 MCP 服务器可视测试工具，简化调试流程。

## 3. 核心功能

- 🔧 MCP 服务器连接测试
- 📊 请求/响应可视化
- 🔌 协议验证
- 🧪 工具调用测试
- 📝 消息检查

## 4. 本地部署

```bash
# 安装
npm install -g @modelcontextprotocol/inspector

# 启动
mcp-inspector
```

## 5. 效果展示

- GitHub Stars：⭐ 8,900
- 官方维护
- 支持所有 MCP 协议版本

## 6. 优缺点

✅ 官方维护，质量有保证
✅ 可视化界面直观
✅ 支持协议验证

⚠️ 需要 Node.js 环境
