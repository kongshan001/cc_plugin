## 4. 本地部署

```bash
# 克隆仓库
git clone https://github.com/conorluddy/ios-simulator-skill ~/.claude/plugins/ios-simulator-skill

# 安装依赖
cd ~/.claude/plugins/ios-simulator-skill
pip install -r requirements.txt
```
