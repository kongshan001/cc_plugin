# Vibetest-Use - 自动化 QA 测试工具

> ⭐ 771 Stars · Python · Browser-Use

## 1. 背景需求

传统的自动化测试需要大量手工编写测试用例，开发者需要 AI 驱动的自动化 QA 测试工具。

## 2. 目标

使用 Browser-Use 代理实现自动化 QA 测试，降低测试成本。

## 3. 核心功能

- 🤖 AI 驱动测试生成
- 🖥️ 浏览器自动化
- 👁️ 视觉回归测试
- 📊 测试报告

## 4. 本地部署

```bash
# 安装
pip install vibetest-use

# 使用
from vibetest import VisualTester

tester = VisualTester()
tester.run_visual_tests("https://example.com")
```
